# 🧾 계약서 리스크 하이라이터

> **복잡한 계약서, 위험 조항은 우리가 찾아드립니다.**  
> 사회 초년생의 권리를 지키는 AI 기반 계약서 리스크 분석 서비스


## 💡 프로젝트 개요

알바 또는 단기 고용 계약서를 처음 접하는 **사회 초년생**들은  
법률 지식 부족과 심리적 부담으로 인해 **부당한 계약 조건을 놓치기 쉽습니다.**

본 서비스는 **계약서 이미지를 업로드하기만 하면**,  
Vision 기반 모델이 계약서를 분석하고 **법적으로 위험한 조항을 하이라이트**,  
더 나아가 **즉시 활용 가능한 수정 요청 스크립트**까지 제공합니다.

> 사인하기 전에 **당당하게 권리를 챙길 수 있도록** 돕는 서비스입니다.

## 🎯 핵심 목표

| 목표 | 설명 |
|---|---|
| ✔ 리스크 시각화 | 원문에 **직관적인 하이라이팅**으로 위험 조항 표시 |
| ✔ 근거 기반 피드백 | 노동법 및 **표준근로계약서 대비 법적 리스크 진단** |
| ✔ 실행 유도 | **복붙 가능한 커뮤니케이션 스크립트** 제공 |

## ⚙️ 기술 아키텍처

본 프로젝트는 2인 협업 구조로 진행되며,  
핵심 엔진은 **Vision LLM + RAG 기반 리스크 분석 로직**입니다.

| 구분 | 기술 | 설명 |
|:---:|:---|:---|
| Frontend | **Streamlit (Replit)** | 모바일 친화적 분석/시각화 UI (**Member B: 조앵**) |
| Core Logic | **Gemini 1.5 Pro (Vision LLM)** | OCR + 조항 구조 분석 + PII 자동 마스킹 (**Member A: 클레버**) |
| 법리 분석 | **RAG** | 표준 근로계약/근로기준법 문서 기반 근거 조회 |
| 협업 | GitHub · VS Code · Replit | 역할 분담 및 버전 관리 |

---

## 🔍 Vision + RAG Flow

1. 사용자 계약서 사진/스캔본 업로드  
2. Vision LLM  
   - OCR + 조항 구조 이해 + 개인정보 자동 마스킹  
3. 텍스트로 추출된 조항을 벡터화  
4. RAG 검색  
   - 표준계약서 + 노동법 DB에서 근거 문서 매칭  
5. 위험도 산정 및 근거 생성  
6. Streamlit UI로 결과 시각화 및 스크립트 제공
