# 🧾 계약서 위험 탐지기

> **복잡한 계약서, 위험 조항은 우리가 찾아드립니다.**  
> 사회 초년생의 권리를 지키는 AI 기반 계약서 리스크 분석 서비스

---

## 😣 Pain Point — 왜 이 서비스가 필요한가?

알바 · 단기 계약을 맺는 사회 초년생들은…

- 📃 **법률 용어 가득한 계약서**를 제대로 이해하기 어렵고
- 😥 불리한 조항이 있어도 **사인 직전에 묻기 어려워** 그냥 넘어가며
- ⚠️ 그 결과, **금전적 피해 / 부당 대우**로 이어지는 일이 빈번합니다.

> “사인하기 전에 문제를 알았더라면… ”  
> 뒤늦게 후회하는 상황을 막고자 합니다.

---

## 🚀 Solution — 우리의 접근 방식

📌 **Vision + LLM + RAG** 기술을 활용하여  
계약서를 *읽고, 이해하고, 액션까지 안내합니다.*

### 🧠 계약서 자동 분석 (Vision + LLM)
- 사진만 찍으면 계약서 문맥까지 이해하고 자연어로 설명
- 핵심 권리/의무 항목을 쉽게 정리하여 제공

### ⚖️ 법적 표준 vs 실제 조항 비교 (RAG)
- 근로기준법 및 표준근로계약서와 비교
- 🚨 **위험하거나 누락된 조항을 하이라이트**
- 명확한 법적 근거까지 함께 표시

### 🗣️ “이 조항, 이렇게 말하세요”
- **사장님께 요구할 수 있는 커뮤니케이션 스크립트** 제공  
  예:  
  _“해당 조항은 근로기준법 제○조에 따라 수정 요청드립니다.”_

## ✨ 핵심 기능 요약

| 기능 | 설명 |
|------|------|
| 계약서 이미지 분석 | 사진 업로드만으로 자동 문서 이해 |
| 위험 조항 탐지 | 비표준/불리 조항 하이라이트 |
| 법적 근거 제시 | 근로기준법 기반 신뢰성 보장 |
| 액션 스크립트 | 사용자 대신 **말해주는 조정 문구** 제공 |
| 안전한 기록 저장 | 개인정보/문서 데이터 보안 처리 |
